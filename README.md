# RMUTI Surin Loan Management System

ระบบบริหารลูกหนี้เงินยืม สำหรับมหาวิทยาลัยเทคโนโลยีราชมงคลอีสาน วิทยาเขตสุรินทร์

## Tech Stack

- Frontend: Next.js, React, Tailwind CSS
- Backend: Node.js, Express, TypeScript
- Database: MySQL
- Authentication: NextAuth.js, JWT

## Workspace

- `frontend/` - Web UI และ NextAuth client-facing app
- `backend/` - REST API, business logic, database access
- `database/` - SQL migrations, seed data, database notes
- `docs/` - System analysis and project documentation
